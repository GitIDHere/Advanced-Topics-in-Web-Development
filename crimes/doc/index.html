Part 5: Documentation

One of the first challenges which I faced during this assignment was converting the CSV data into a well-formed XML file. With this, I initially struggled to structure my PHP code in a way which read the CSV data that I wanted. This was because after converting the provided XLS file into the CSV, the CSV contained additional data, as well as data positioned in an unorganised way. I therefore had to delete some of the data and reposition them so that I knew which line I was reading in. I was then able to structure my PHP to read the data I wanted and create an XML string based on the data that was read.

Another challenge which I faced was during the implementation of the PUT request. Since this request required me to create an XML element representing a new area, I struggled to find a way which did not require me to hardcode the new XML area elements within PHP, since doing so would result in bloated code. After much thought, I finally found that by using an XML file containing a template of an empty area, I would be able to read that template within PHP to create and populate the new area with the data passed in from the request. This provided a solution which did not require me to hardcode the new area XML, and also it allows for people to change the XML area template without having to access my code.

Client side visualisation also gave me some difficulties on the use of Chart.js to generate the bar chart. Because of my minimal knowledge and experience with JQuery, the data array structure I created for the bar chart was malformed. This lead the bar chart scales and labels to be squashed together and displayed inaccurately. However, after asking for help on Piazza, I was guided on the right direction by a fellow peer on as to what might be causing this to happen. As it turned out, it was the structure of elements within the array I was using for the bar chart.

From this assignment, the challenges that I faced also enabled me to learn new coding approaches and methods. For example, this was the first time I had used PHP in a way which combined both procedural and object-oriented approaches to coding. Also, this was the first time I had made use of JSON significantly. This allowed me to learn how to correctly structure a JSON response, as well as learning how it integrates within JavaScript.

Overall, this assignment has been a long journey for me; allowing me to learn and experience new approaches to creating a RESTful API. I was provided with minimal resources and requirements which allowed me to pave my own path on as to how I would tackle the assignment. The challenges that I faced along the way helped me to realise what I could to improve my code, as well as the approaches that I was taking.
